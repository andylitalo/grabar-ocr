Grabar Line Labeler

Background jobs

Draw one box per region, tight to the text. Pick a region's type; add single-column / header bands as needed. Drag a corner to resize, the inside to move. Segment lines slices the page for manual transcription; Label and Translate also queues OCR + translation in the background.

Each line is pre-seeded with the detector's guess ( = detector says non-character). Fix only the disagreements, then submit.
